void Doodlebug::simpleMove ( )

simpleMove() attempts to move a doodlebug into an adjacent cell, choosing one randomly and trying the rest. If none of the cases which were tried work, move() calls itself recursively to try again. It also checks whether it is on the edge of the grid, and terminates if the ant is unable to move
